## 🎯 الخدمات المُنفذة

### 1. Classifier Service (`class-classifier.php`)

#### مصفوفة الجهات الفاعلة (Actors Matrix)
```php
[
    // المحليون
    'lb_army' => 'الجيش اللبناني',

    'sy_gov' => 'الحكومة السورية',

    'pal_fatah' => 'حركة فتح',
    
    // الدوليون

    'nato_org' => 'حلف الناتو',
    'brics_org' => 'دول البريكس',

```

#### دوال التصنيف الرئيسية:

1. **`extractNamedNonMilitaryActor($text)`**
   - استخراج الجهات المسماة من النص
   - نظام ترجيح متقدم (Keyword Weighting)
   - مكافأة للذكر في بداية النص (Subject Position Bonus)
   - حل التعارضات بين الجهات المتعددة

2. **`contextMemoryInfer($text, $currentActor)`**
   - الاستدلال من سياق النص عند عدم وجود ذكر صريح
   - خرائط قرائن سياقية لكل جهة
   - دعم الذاكرة السياقية

3. **`governorAI($currentResult, $text)`**
   - الحاكم الذكي لاتخاذ القرار النهائي
   - دمج الاستخراج الصريح والاستدلال السياقي
   - تحديد مستوى الثقة (high/medium/low)
   - التحقق من التناسق بين المنطقة والجهة

4. **`forceRequestedActorRule($forcedActorId, $currentResult)`**
   - فرض جهة معينة يدويًا أو بقاعدة محددة
   - دعم التجاوز اليدوي للتصنيف التلقائي

#### مصفوفات التصنيف:

| المصفوفة | القيم |
|----------|-------|
| **المستويات الاستراتيجية** | عام، عسكري، أمني، سياسي، اقتصادي/لوجستي، إعلامي، الدبلوماسي والسياسي |
| **المستويات التكتيكية** | عملياتي، تكتيكي، ميداني |
| **المناطق** | لبنان، فلسطين، الأراضي المحتلة، سوريا، العراق، إيران، الخليج، الولايات المتحدة، أوروبا، دولي |
| **الأسلحة** | صواريخ بالستية، كروز، طائرات مسيرة، طيران حربي، مدفعية، أسلحة رشاشة، قطع بحرية، هجوم إلكتروني، عبوة ناسفة |
| **النيايا** | هجوم، دفاع، جمع معلومات، تصريح، مفاوضات، تهديد، تدريب، تموضع/رسائل سياسية، نشاط ميداني، حرب نفسية |

### 2. Newslog Service (`class-newslog.php`)

#### هيكل حقول التصنيف:
```php
[
    'title' => ['label' => 'العنوان', 'type' => 'string', 'required' => true],
    'strategic_level' => ['label' => 'التصنيف الاستراتيجي', 'type' => 'enum'],
    'tactical_level' => ['label' => 'التصنيف التكتيكي', 'type' => 'enum'],
    'region' => ['label' => 'المنطقة', 'type' => 'string'],
    'actor' => ['label' => 'الفاعل', 'type' => 'string'],
    'target' => ['label' => 'الهدف', 'type' => 'string'],
    'context' => ['label' => 'السياق', 'type' => 'string'],
    'intent' => ['label' => 'النية', 'type' => 'enum'],
    'weapon' => ['label' => 'السلاح', 'type' => 'string'],
    'outcome' => ['label' => 'النتيجة', 'type' => 'string'],
    'source' => ['label' => 'المصدر', 'type' => 'string'],
    'points' => ['label' => 'النقاط', 'type' => 'integer'],
]
```

#### الوظائف الرئيسية:

1. **`isManualLockedRow($row)`**
   - التحقق مما إذا كان الصف مقفلًا يدويًا
   - منع التعديل التلقائي على التصنيفات المُراجعة

2. **`extractClassificationFields($row)`**
   - استخراج الحقول المهيكلة من صف الأخبار
   - دعم التنسيق النصي غير المهيكَل
   - التحقق من صحة القيم

3. **`applyManualOverride($rowId, $fields)`**
   - تطبيق تصحيح يدوي على التصنيف
   - حفظ الطابع الزمني ومعرف المستخدم
   - تكامل مع WordPress Post Meta

4. **`validateClassificationData($fields)`**
   - التحقق من اكتمال وصحة البيانات
   - إرجاع قائمة بالأخطاء والتحذيرات

5. **`buildDisplayString($fields)`**
   - إنشاء نص عرض بصيغة قابلة للقراءة
   - مثال: "استراتيجي: عام\nتكتيكي: عملياتي\nمنطقة: إيران"

6. **`compareClassifications($old, $new)`**
   - مقارنة نسختين من التصنيف
   - إظهار الفروقات بدقة

```php
use Beiruttime\OSINT\Services\Newslog;
```

## 📋 حالات الاختبار المُغطاة

### من البيانات المُقدمة (100+ حالة):

1. **مفاوضات إيران-أمريكا في باكستان**
   - السياق: دبلوماسي/سياسي
   - الجهات: إيران، الولايات المتحدة، باكستان
   - التحدي: تعدد الجهات في نفس النص
   - الحل: `resolveConflictingActors()` مع كشف سياق المفاوضات

2. **عمليات الاقتحام في الضفة الغربية**
   - السياق: أمني/عسكري
   - الجهة: جيش العدو الإسرائيلي
   - المنطقة: فلسطين
   - النية: هجوم

3. **الغارات الجوية على لبنان**
   - السياق: عسكري
   - الجهة: جيش العدو الإسرائيلي
   - السلاح: طيران حربي
   - المناطق المتعددة: بنت جبيل، صور، النبطية...

4. **تصريحات المسؤولين الإيرانيين**
   - السياق: تصريحي/إعلامي
   - الجهة: الحكومة الإيرانية / حرس الثورة
   - الهدف: مضيق هرمز، إسرائيل...

5. **عمليات المقاومة الإسلامية**
   - السياق: عسكري
   - الجهة: المقاومة الإسلامية
   - السلاح: صواريخ، مسيّرات
   - الهدف: ثكنات، آليات عدوة
